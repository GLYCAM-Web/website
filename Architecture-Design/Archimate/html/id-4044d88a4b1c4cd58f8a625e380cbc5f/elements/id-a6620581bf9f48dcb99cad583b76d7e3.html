Determine Implied Services ()
Inspect the JSON object to see if any Services are directly implied at the user-friendly level of the interface. Here, directly implied means that the user has requested the service. Dependencies are dealt with elsewhere. That is, if someone sends just Entity=Sequence and Payload=DManpa1-OH, then the implied service is Build3DStructure. Another step will figure out whether or not the sequence should be validated and evaluated first.
